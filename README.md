# Blog Website

> Đây là một trang web fullstack blog.

## Công nghệ sử dụng:

- ReactJS, Tailwind CSS, Firebase, Redux Toolkit, React Hook Form, Flowbite React.

- NodeJS, ExpressJS, MongoDB, Mongoose.

## Chức năng:

- Tìm kiếm bài viết, có thể bình luận và chỉnh sửa bình luận ở mỗi bài viết.

- Người dùng có thể đăng ký tài khoản mới, đăng nhập hoặc chỉnh sửa thông tin.

- Sử dụng Firebase để đăng nhập google và lưu trữ hình ảnh.

- Quản trị viên có thể tạo mới, chỉnh sửa và xóa bài viết.

- Quản trị viên có thể quản lý người dùng, bình luận.

- Sử dụng Redux Toolkit lưu trữ thông tin người dùng và dark mode.
